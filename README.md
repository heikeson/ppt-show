# PPT SHOW - 互动展示系统

基于 Web 技术的 PPT 展示工具，支持从 URL 加载 PPT 并在浏览器中展示，支持视频播放、实时批注、手势滑动及沉浸模式。

## 功能特性

### 基础功能
- **PPT 展示**：从远程服务器加载 PPT 图片并展示
- **翻页控制**：支持键盘左右键、点击边缘区域、鼠标滚轮、手势滑动翻页
- **页码跳转**：可直接输入页码跳转到指定页面
- **全屏模式**：按 Enter 键进入全屏沉浸式演示

### 批注系统
- **画笔工具**：在 PPT 画面上实时绘制批注
- **橡皮擦**：清除错误的批注内容
- **颜色选择**：支持红色、蓝色、绿色、黄色、白色五种预设颜色及自定义颜色
- **粗细调节**：可调整笔触粗细（1-20px）
- **撤销/重做**：支持 Ctrl+Z 撤销、Ctrl+Y 重做
- **清空功能**：按 Ctrl+D 清空当前页面所有涂鸦

### 空白画板
- **纯净白板模式**：按 B 键进入精美网格背景的白板模式
- **独立绘制**：白板模式下绘制内容与 PPT 批注独立保存

### 视频支持
- **视频播放**：支持在特定页面嵌入视频播放（按 V 键触发）
- **独立播放器**：视频在全屏容器中播放，支持原生控件

### 沉浸模式
- **自动隐藏 UI**：3 秒无操作后自动隐藏所有控制元素
- **画笔模式保持可见**：使用画笔时 UI 保持显示

### 配置选项
- **远端资源路径**：可配置 PPT 图片的远程服务器路径
- **预加载策略**：支持极速模式（前后 2 页）、均衡模式（前后 3 页）、激进模式（前后 5 页）

## 使用方式

### URL 参数配置

通过 URL 参数传递配置信息：

```
index.html?url=Base64编码的资源路径
```

例如：
```
index.html?url=aHR0cHM6Ly9leGFtcGxlLmNvbS9wcHQtaW1ncy8=
```

其中 `aHR0cHM6Ly9leGFtcGxlLmNvbS9wcHQtaW1ncy8=` 是 `https://example.com/ppt-imgs/` 的 Base64 编码。

### 配置文件格式

服务器端需要提供 `imgs.txt` 配置文件，格式如下：

```
总页数
图片1名称|视频1名称（可选）
图片2名称|视频2名称（可选）
...
```

示例：
```
5
slide1.jpg
slide2.jpg|video1.mp4
slide3.jpg
slide4.png
slide5.jpg
```

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← / → | 上一页 / 下一页 |
| Space | 下一页 |
| Enter | 全屏切换 |
| Escape | 退出全屏 / 关闭面板 |
| P | 画笔模式开关 |
| E | 橡皮擦模式开关 |
| S | 笔刷设置面板 |
| B | 空白画板模式 |
| V | 播放视频 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+D | 清空当前页 |

### 触屏手势

- **左滑**：下一页
- **右滑**：上一页
- **点击左边缘 1/5 区域**：上一页
- **点击右边缘 1/5 区域**：下一页
